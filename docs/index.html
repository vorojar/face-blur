// Features

每一帧都按
人脸的形状打码。

01 / 06

双模型检测

FaceLandmarker 精准轮廓 + FaceDetector 侧脸兜底。正脸、侧脸、戴口罩、低头——不漏脸。

02 / 06

468 点轮廓打码

沿面部关键点拟合多边形，不是粗暴矩形。头发、背景、脖子都保留，只糊脸。

03 / 06

边缘羽化

alpha 渐变混合，打码区域与原图自然过渡。零硬边，和原始画面融为一体。

04 / 06

硬件加速编码

自动识别 VideoToolbox / NVENC / QSV / AMF，直走 GPU 编码器。macOS 5×、NVIDIA 8× 实时。

05 / 06

管道直出

原始帧通过 pipe 直接喂 FFmpeg，无中间文件、无二次编码。画质无损、磁盘干净。

06 / 06

零配置运行

首次运行自动建 .venv、装依赖。一行命令，不用懂 Python 环境。

// Quick Start

系统有 Python 3 和 FFmpeg，剩下的它自己来。

把 face_blur.py 丢到视频同目录，一行命令跑起来。没有依赖预装步骤——工具会自己创建虚拟环境、拉 mediapipe 和 opencv-python、启动硬件编码器。

  • Python 3.10+
  • FFmpeg（系统安装）
  • 2.2 MB 脚本 · 无 Docker · 无外部服务
bash · default (gaussian blur) 
# 克隆仓库
$ git clone https://github.com/vorojar/face-blur.git
$ cd face-blur

# 直接跑（首次运行自动装依赖）
$ python3 face_blur.py your_video.mp4

# 马赛克模式 + 加大打码范围
$ python3 face_blur.py video.mp4 --mode mosaic -p 0.5

// CLI parameters

8 个参数，按需调。

参数说明默认值
-o, --output输出文件路径输入文件名_mosaic.mp4
-s, --strength模糊强度，越大越模糊80
-p, --padding人脸轮廓外扩比例0.3
--modegaussian（高斯模糊）/ mosaic（马赛克）gaussian
--min-face-size人脸最小边长（像素），低于此值不打码40
--detect-interval每 N 帧检测一次（提速）3
--min-confidence人脸检测最低置信度0.3
--preview实时预览处理效果—
// Pipeline

一条流水线，
没有中间产物。

视频输入 │ ▼ OpenCV 逐帧读取 │ ├─ MediaPipe FaceLandmarker → 468 点人脸轮廓（正脸 / 微侧脸） ├─ MediaPipe FaceDetector → 矩形框转椭圆（侧脸兜底） ├─ 去重合并 │ ▼ 高斯模糊 + 羽化 mask + alpha 混合 │ ▼ 管道 pipe → FFmpeg 硬件加速编码 + 原音轨 copy │ ▼ 输出 MP4
